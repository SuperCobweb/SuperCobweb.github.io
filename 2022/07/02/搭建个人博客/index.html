搭建个人博客

发表于 2022-07-02 分类于 网站 阅读次数：

用hexo搭建个人博客比自己写一个网站来的轻松

优点：纯静态文件，不依赖服务器和数据库

缺点：对笔者电脑环境的依赖过大，换一台电脑需要重新配置环境

我使用Hexo+github&云服务的模式部署的。

总的来说单纯的将博客放到github上比较容易实现，但由于国内网络环境访问不便还有本人有一台闲着的云服务器，遂将网站也在放在了云服务上。

Github篇

  1. 先再github上创建一个仓库，注意仓库名字是固定的，为github账号名.github.io

  2. 自己电脑上下载node.js

  3. 自己电脑上下载Git并配置

    生成rsa秘钥，把公钥配置到github上，这样以后不需要每次登录github时输入密码
    在cmd窗口输入
    ssh-keygen -t rsa
    直接回车就行，全部选择默认
    密钥生成成功，打开文件夹 C:\Users\Cobweb.ssh下的id_rsa.pub文件，里面就是本机的rsa密钥。打开你的GitHub，点击你的头像，依次选择setting -> SSH and GPG keys -> New SSH key ，将你电脑的rsa密钥添加到GitHub上。
